Inspetor de elementos

É uma ferramenta do desenvolvedor que permite visualizar o código fonte de qualquer página web, facilitando para estudo ou até mesmo servir como base em projetos web de quem estiver inspecionando.

Quem oferece esta ferramenta é o próprio navegador como o CHROME, OPERA, FIRE FOX, SAFARI etc. Basta apenas clicar na tecla F12 ou indo no canto superior direito onde vai ter três pontos, mais informações, ferramentas do desenvolvedor.

Estrutura básica html

A página principal sempre vai ser chamada de index.html (pode colocar outros nomes, mas, index sempre vai ser padrão).

Html consiste em três partes HTML, HEAD E BODY., tudo que estiver entre BODY é o que vai aparecer para o usuário, HEAD são coisas que vai carregar antes de exibir para o usuário.

Tags

Html, head, body também são uma tag!.

< i > significa itálico, todo item que estiver dentro desta tag vai o navegador irá entender que a palavra está em itálico.

< strong >é uma tag utilizada para deixar o texto em negrito.

Também tem como combinar as tags, por exemplo, caso queira deixar alguma parte do texto em itálico e negrito também é só colocar = < strong > < i > qualquer texto < /i >< /strong >.

Algumas tags não tem fechamento como INPUT ou IMG, ou seja, não tem início nem um fim, é simplesmente um campo.

Atributo das tags

A tag < strong >tem um atributo chamado id. Ele serve para identificar no javascript qual elemento você quer mexer. Resumidamente é um identificador.

Dentro da tag < strong > também tem o atributo STYLE que serve para colocar comandos CSS direto em um elemento.

Temos também o < input >, que vai basicamente fazer um campo para texto em seu html, que é muito utilizado para formulários onde vão ter campos para digitar, e ele tem o atributo < type > onde você pode alterar o campo de digitação para somente números, textos ou até mesmo cores.

Já o < IMG >tem o atributo SRC que vai permitir colocar um endereço de imagem em seu html e deve aparecer em sua página, pode ser uma imagem da internet ou até mesmo uma imagem dentro de sua pasta salva em sua área de trabalho. E para complementar temos o atributo WIDITH que seria a largura da imagem que você escolheu, lá você irá escolher a numeração que vai representar a largura da imagem e ira aparece em sua página principal.

Textos

Sobre textos temos os < H > que vão definir seus títulos em seu html, como o < H1 > que seria o título, < H2 > subtítulo e assim por diante.

Temos a tag < P > de parágrafo, como já diz a tag ela serve para fazer um campo somente para o parágrafo que você deseja colocar seu texto em seu html.

A tag < BLOCKQUOT > serve para você colocar alguma citação em seu texto, seja uma frase de efeito, uma fala de algum autor etc.

A tag < U > serve para sublinhar alguma palavra de seu texto, colocando esta tag em alguma palavra em especifico de seu texto ir ficar sublinhado

Você pode também usar a tag < MARK > em alguma palavra de seu texto para deixar com marca texto em sua página.

Listas ordenadas e não ordenadas

< OL > é uma tag que permite você criar uma lista em ordem do 1,2,3,4 e assim por diante, mas é uma tag que precisa de outra tag “filha” que seria uma tag dentro da outra que é a < LI > que vai poder fazer um campo onde você vai poder colocar o item da lista.

< UL > é uma lista que não vai ter uma ordem como a < OL > e vai fazer uma lista com uma bolinha indicativa ao invés de uma numeração indicando uma ordem, ela funciona da mesma forma que a < OL >, tendo que usar a tag < LI >como uma tag filha para poder adicionar um item à lista.

Links

(voltar)

Para fazer as navegações em nosso site utilizaremos a tag < a href= “SEU LINK” > e junto vai ter um espaço em antes de fechar a tag < a > que seria para colocar um nome no seu link.

Você pode usar esta tag até mesmo para fazer navegações em seu próprio site também.
